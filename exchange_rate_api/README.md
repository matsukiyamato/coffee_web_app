ブランチ「exchange_rate_api」

内容
────────────────────────────────────────
為替レートAPIで今の為替を取得（円 / ドル）
取得した値をCSVファイルに保存（年-月-日　時間:分:秒,為替値）


実行方法
────────────────────────────────────────
1.　.envを開く
2.　app.exchangerate-api.com　にて自分のAPIを取得する
3.　your_api_key_here　の部分を自分のAPIを貼る
4.　exchange_rate.py　を実行する


ネットワークエラーの場合
→WI-FI接続にする
